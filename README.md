# AquariumMonitor
ESP8266で水槽の状態をモニタする話
## 仕様

*水温、室温、湿度のモニタ
*センサデータのmunin-nodeでの出力機能
*LED、CO2、ブクブクのON/OFF
*水温閾値設定による冷却ファン、エアコンON/OFF
*Web設定インターフェース
